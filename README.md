# SkillSwap 🔄
### Pakistan's Peer-to-Peer Skill Exchange Platform

> Trade your skill. Learn theirs. Zero rupees.

**[🔗 Live Demo](https://skillswap-ten-lilac.vercel.app/)**

---

## What is SkillSwap?

Pakistani students and professionals have skills others need — 
and need skills others have. But courses are expensive. 
SkillSwap solves this: trade what you know for what you want 
to learn. No money. Just skills.

**Figma for Python. Photography for SEO. Design for Code.**

---

## Features

- 🎯 **Live Skill Listings** — browse real swap offers with 
  category, city and level filters
- 🔍 **Smart Search** — search by skill, name or city in real time
- 👤 **Profile Modals** — view full profiles with ratings, 
  completed swaps and skill tags
- 🔄 **Swap Request Flow** — send requests with custom messages
- 📊 **Dashboard** — manage incoming/active/completed swaps, 
  notifications and smart matches
- ✨ **Post a Swap** — form that actually adds your card to the grid
- 📱 **Fully Responsive** — mobile, tablet and desktop

## Tech Stack

| | |
|---|---|
| Language | HTML5, CSS3, Vanilla JavaScript (ES6+) |
| Animations | CSS 3D Transforms, IntersectionObserver, Canvas API |
| Design | Sora + Inter fonts, warm dark theme, custom SVG logo |
| Storage | No backend — pure frontend |

---

## Design Highlights

- Custom SVG logo with animated sparkles
- 3D orbiting skill tag cloud in hero (pure CSS)
- Cursor-tracked card tilt with specular shine effect
- Live swap ticker showing real-time activity
- Animated counters, staggered reveals, spring animations
- Lahore Bazaar aesthetic — warm darks, amber/yellow accent
